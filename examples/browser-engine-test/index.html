web-csv-toolbox - Engine Test Suite

Purpose: Test and compare different parsing engines (JavaScript, WASM, Worker, GPU).
Usage: Click individual test buttons or "Run All Tests" to execute tests.
Automated Testing: Use Playwright with pnpm test to run headless tests.

JavaScript Engine (stable)

Pure JavaScript implementation. Maximum compatibility, runs on main thread.

WASM Engine

WebAssembly-based parsing. Faster than JS, runs on main thread.

Worker Engine (recommended)

WASM in Web Worker. Non-blocking UI, good performance.

GPU Engine (turbo)

WebGPU-accelerated parsing. Maximum speed for large files.
